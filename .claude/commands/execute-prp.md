# Execute PRP

지정된 PRP 파일을 읽고 단계별로 구현한다.

## 프로세스

**Phase 1 - 이해**
- PRP 파일 전체 읽기
- 관련 파일/패턴 파악
- 불명확한 부분 질문

**Phase 2 - 계획**
- 구현 순서 확정
- 영향받는 파일 목록 작성

**Phase 3 - 구현**
- PRP 순서대로 코드 작성
- 기존 패턴 유지
- 파일 500줄 이하 유지

**Phase 4 - 검증**
- 검증 게이트 체크리스트 실행
- 실패 시 수정 후 재검증

**Phase 5 - 완료**
- PRP 체크리스트 전체 확인
- 완료 보고
