## Design Context

### Users
소규모 식당 사장님들 (캐나다 이민자 중심). 기술 친숙도 낮음. 주로 모바일(PWA)로 사용. 바쁜 주방/홀 환경에서 빠르게 확인. 영어가 모국어가 아닐 수 있음.

### Brand Personality
친근함 · 명확함 · 믿음직함  
전문적이되 딱딱하지 않게. "동네 잘 나가는 식당 사장이 믿고 쓰는 도구" 느낌. 복잡함 없이 바로 이해되는 UI. 은행 앱처럼 차갑지 않고, 장난감처럼 가볍지도 않음.

### Aesthetic Direction
- **Theme**: 라이트 모드. 밝고 눈이 편한 배경, 과하지 않은 색 대비.
- **Feel**: 따뜻한 중립 톤. 순백색/차가운 회색보다 약간 warm-tinted 베이지/오프화이트.
- **Accent**: 음식/식당 산업과 어울리는 따뜻한 색 (과하지 않게). 현재 `#2563eb` corporate blue 탈피.
- **Anti-reference**: 일반 SaaS 대시보드 (차갑고 파란 generic Tailwind 템플릿).

### Design Principles
1. **즉시 이해** — 아이콘/색상/레이블이 설명 없이 의미 전달
2. **전문성 = 일관성** — 같은 패턴, 예측 가능한 레이아웃이 신뢰감 줌
3. **따뜻한 중립** — 식당 업장 분위기와 어울리는 오프화이트/웜톤 팔레트
4. **정보 계층** — 중요한 숫자/상태가 한눈에 들어와야 함
5. **모바일 최우선** — 터치 타겟, 읽기 쉬운 폰트 크기, 손가락으로 쓰는 UI
